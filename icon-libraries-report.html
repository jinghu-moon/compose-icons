通用 UI 图标库
Lucide 1,500+ 图标 ISC 推荐
lucide.dev ↗

Feather Icons 的社区分支，也是目前最活跃的线条图标库之一。所有图标统一绘制在 24×24 网格上，风格极度一致，线条克制、留白充足。与 Feather 相比，Lucide 持续维护且图标数量是其数倍。原生支持 React、Vue、Svelte、Angular 等主流框架，并提供 Figma 插件。Tree-shaking 支持完善，按需引入不增加包体积。

线条风格 SVG React / Vue / Svelte Figma 插件 活跃维护
Heroicons 450+ 图标 MIT
heroicons.com ↗

由 Tailwind CSS 团队精心制作，与 Tailwind 生态深度集成。提供 Outline（线条）、Solid（填充）、Mini（20px）、Micro（16px）四种变体，覆盖小尺寸到标准尺寸的所有使用场景。图标数量虽不多，但每一枚都经过仔细打磨，风格高度统一，非常适合追求精致感的产品 UI。

Tailwind CSS 4 种变体 React / Vue 精品小集
Phosphor Icons 9,000+ 图标 MIT 推荐
phosphoricons.com ↗

在数量与质量上都极为出色的开源图标库，最大亮点是每个图标都有 Thin、Light、Regular、Bold、Fill、Duotone 六种变体风格，设计师可以在不换库的情况下随意切换视觉重量。Duotone（双色）变体尤其适合需要层次感的现代界面。支持 Web、React、Vue、Flutter、Elm 等多平台。

6 种变体 Duotone Flutter 数量最多之一
Tabler Icons 5,600+ 图标 MIT
tabler.io/icons ↗

来自 Tabler UI 框架团队，基于 24×24 网格绘制的像素级精准 SVG 图标集。风格简洁现代，线条粗细可调（stroke-width 参数化），是线条图标中数量最丰富的库之一。提供 SVG、Web Font、React、Vue 多种使用方式，并支持在线自定义颜色和尺寸后直接下载。

线条风格 Stroke 可调 像素级精准 React / Vue
Feather 286 图标 MIT
feathericons.com ↗

极简主义线条图标的鼻祖，奠定了现代线条风格图标的审美基准。每个图标严格遵循 24×24 网格、2px stroke、圆角线帽的统一规范，风格高度克制。虽然图标数量仅有 286 个，但覆盖了绝大多数通用 UI 场景。如今已被 Lucide 等库继承并扩展，但 Feather 本身仍是极简风格的标杆参考。

极简主义 经典标杆 轻量 SVG
Material Symbols 3,000+ 图标 Apache 2.0 Android 首选
fonts.google.com/icons ↗

Google 官方 Material Design 图标库，也是 Android 开发的官方图标系统。最大特点是基于可变字体技术，支持 Fill、Weight、Grade、Optical Size 四个设计轴动态调节，一套图标可以无缝适配从细线到填充的所有风格。在 Jetpack Compose 中可通过 material-icons-extended 依赖直接引用，无需导入 SVG 文件。

Google 官方 可变字体 Jetpack Compose 4 轴调节 Android 推荐
Material Design Icons (MDI) 7,000+ 图标 Apache 2.0
pictogrammers.com ↗

Material Symbols 的社区扩展版，由 Pictogrammers 社区维护。图标数量远超 Google 官方版本，补充了大量垂直领域图标（智能家居、医疗、气象、游戏等）。风格与官方 Material 保持一致，可与官方库混用。适合需要非常规图标但又想保持 Material 视觉语言的项目。

社区扩展 Material 风格 垂直领域图标 7000+
Bootstrap Icons 2,000+ 图标 MIT
icons.getbootstrap.com ↗

Bootstrap 官方出品，但完全可以独立于 Bootstrap 框架使用。提供 SVG、SVG Sprite、Web Font 三种引入方式，支持 npm 安装。图标风格偏向传统 UI 控件，覆盖了丰富的表单、媒体、导航类图标，线条与填充两种形态并存。对于使用 Bootstrap 生态的项目而言是最自然的选择。

Bootstrap 生态 SVG Sprite Web Font 可独立使用
Remix Icon 2,800+ 图标 Apache 2.0
remixicon.com ↗

定位为"中性系统符号"，强调适用于各种产品设计而不带有强烈的平台风格。每个图标同时提供线条（Line）和填充（Fill）两个版本，分类清晰涵盖媒体、商务、地图、天气等多个领域。设计风格介于 Feather 的纤细与 Material 的厚重之间，是非常平衡的通用选择，免费商用。

中性风格 Line + Fill 免费商用 分类完善
Iconoir 1,600+ 图标 MIT
iconoir.com ↗

以"真正免费"为卖点——无需注册、无付费版本、无邮件墙，所有图标直接可用。风格现代简洁，线条圆润，具有一定的个性化特征，不像部分库那样过于通用。支持 SVG、Web Font、React、React Native、Flutter、Figma、Framer 多平台。适合追求干净审美、同时希望避免 Font Awesome 过度流行感的项目。

真正免费 Flutter React Native 无需注册
Boxicons 1,634+ 图标 MIT
boxicons.com ↗

分为 Regular（线条）、Solid（填充）、Logos（品牌）三大类别，是少数在单一库中同时包含通用 UI 图标和品牌 Logo 的开源方案。圆角设计使整体风格偏向友好、现代。提供 CSS、Web Component、React、Vue 多种集成方式，支持在线定制颜色和尺寸。

Regular + Solid + Logos 圆角风格 Web Component
Ionicons 1,300+ 图标 MIT
ionic.io/ionicons ↗

由 Ionic Framework 团队出品，专为移动端和跨平台应用设计。提供 Outline、Filled、Sharp 三种风格，其中 Outline 和 Filled 对应 iOS 视觉习惯，Sharp 版本线条更硬朗适合 Android/Material 场景。SVG 格式，支持通过 Web Component 在任何框架中使用，也可用于 Ionic / Capacitor 原生应用。

Ionic 出品 移动端优先 Outline / Filled / Sharp 跨平台
Hugeicons 4,000+ 图标 MIT
hugeicons.com ↗

面向现代 SaaS 和产品界面的图标库，设计精细度高，图标在大尺寸下仍有良好的视觉表现。提供 Stroke、Solid、Bulk、Duotone、Twotone 多种变体，每种变体都经过独立调整而非机械复制。有 Figma 官方插件和 React 组件包，是快速增长的新兴库之一，免费商用。

SaaS 风格 多变体 Figma 插件 高精细度
Carbon Icons 2,000+ 图标 Apache 2.0
carbondesignsystem.com ↗

IBM 企业级设计系统 Carbon Design System 的官方图标库。风格严谨、工业感强，在 16×16 和 32×32 像素网格下都有清晰的渲染表现。分类非常完整，特别在数据分析、业务流程、IT 基础设施类图标上有大量其他库欠缺的内容。适合企业级管理后台、数据仪表板等偏工具性的产品。

IBM 出品 企业级 数据分析图标丰富 工业风格
Iconsax 1,000+ 图标 CC0
iconsax.io ↗

最大特点是每个图标提供 Linear、Outline、Broken、Bold、Bulk、TwoTone 六种独立风格，设计师在保持图标语义一致的同时可以自由切换视觉层次。起源于 Figma 社区，拥有大量 UI/UX 设计师用户。对于 React 项目有对应的 NPM 包，使用体验顺畅。

6 种风格 Broken 断点风格 Figma 原生
Streamline Icons 10,000+ (免费部分) 免费 / Pro
streamlinehq.com ↗

体量极大的图标资源平台，涵盖 UI 线条图标、插画风格图标、3D 图标等多个系列。免费部分图标数量依然可观，质量整体较高。有付费的 Pro 订阅版本提供更多高级内容，但对于大多数常规项目免费版已足够。SVG 和 PNG 格式均可下载，同时提供 Figma 和 Sketch 素材包。

体量最大之一 多系列 插画风格可选 部分付费
css.gg 700+ 图标 MIT
css.gg ↗

独特之处在于图标完全由纯 CSS 实现，不依赖任何图片或 SVG 文件。这意味着极低的加载开销，且图标可以通过 CSS 变量随时更改颜色、尺寸，完全响应主题切换。也提供了 SVG 和字体版本供需要的场景使用。适合对性能极度敏感、或需要纯代码方案的特殊场景。

纯 CSS 实现 零图片依赖 极轻量 主题友好
Majesticons 760+ 图标 MIT
majesticons.com ↗

同时提供 Line（线条）和 Solid（填充）两种版本，设计精致克制，细节打磨到位。图标数量适中，但覆盖了常见 UI 场景。整体风格介于 Lucide 的纤细和 Carbon 的厚重之间，适合需要稍有分量感但又不失现代感的产品界面。免费商用，SVG 格式。

Line + Solid 精致克制 免费商用
Circum Icons 1,000+ 图标 MPL 2.0
circumicons.com ↗

分类清晰的开源 SVG 图标库，提供按类别浏览和搜索，免费用于个人和商业项目。风格现代简洁，线条圆润，图标尺寸统一。有 React 和 Vue 组件包，易于集成。相对年轻的库，仍在持续增长中，适合对 Lucide 风格有兴趣但希望尝试不同选择的项目。

分类完善 React / Vue 免费商用
Octicons 500+ 图标 MIT
primer.style ↗

GitHub 官方设计系统 Primer 的图标库，专为代码协作和开发工具场景设计。包含大量与 Git、仓库、PR、Issue、代码审查相关的专有图标，这在其他库中很难找到。提供 16px 和 24px 两种规格，支持 Ruby、Jekyll、React 等集成方式。非常适合构建开发工具类产品或 GitHub 风格界面。

GitHub 出品 开发工具图标 Git 专有图标 Primer 设计系统
Font Awesome 2,000+ (免费) 免费 / Pro
fontawesome.com ↗

Web 图标领域的老牌标准，自 2012 年发布以来已被数千万网站采用。免费版包含 Solid、Regular、Brands 三种风格共 2000+ 图标，付费 Pro 版提供更多变体和图标。由于使用极其广泛，其图标形状已成为许多用户对"图标应该长什么样"的心理预期，但也正因如此，视觉辨识度略显通用。

历史最悠久 使用最广泛 Pro 付费扩展 包含品牌图标
Line Awesome 1,400+ 图标 MIT
icons8.com/line-awesome ↗

由 Icons8 打造的 Font Awesome 替代品，目标是以一行代码从 Font Awesome 无缝切换过来。提供与 Font Awesome 图标一一对应的线条版本，风格更轻盈现代。适合已有大量 Font Awesome 使用代码、希望切换到更清晰线条风格但又不想大幅重构的项目。

FA 平替 线条风格 一行代码切换
品牌 Logo 图标库
使用品牌图标时需注意：大多数品牌的 Logo 受商标法保护，开源授权仅适用于文件格式本身，不代表品牌使用授权。用于开发工具导航、技术栈展示、平台标识等场景通常没问题，但不建议用于商业宣传材料中对品牌的直接指代。
Simple Icons 3,000+ 品牌 CC0 品牌首选
simpleicons.org ↗

最权威的开源品牌图标库，社区活跃，更新频率极高。每个图标都经过严格审核以确保与官方品牌形象一致，并附带官方十六进制品牌色，方便直接应用品牌配色。采用 CC0 协议（接近公共领域），是授权最宽松的品牌图标来源。提供 npm 包，支持 CDN 引入，有丰富的第三方框架集成生态。

覆盖最广 含官方品牌色 CC0 协议 高频更新 npm
Devicons 500+ 技术品牌 MIT
devicon.dev ↗

专注于编程语言、开发框架、工具和平台的 Logo 图标库。涵盖从 Python、Rust、Go 等编程语言到 Docker、Kubernetes、Figma 等工具的完整图标集。每个图标通常提供 Original（官方配色）、Plain（单色）、Line（线条）等多个变体，以及 Wordmark（带文字）版本，非常适合技术栈展示页面和开发者简历。

编程语言 开发工具 多变体 含 Wordmark
SVG Porn (svgporn) 1,000+ 品牌 CC0
svgporn.com ↗

高质量品牌 SVG 的聚合集合，注重图标的视觉精准度，力求还原品牌官方 Logo 的细节。直接提供 SVG 文件下载，无需注册，CC0 协议。相比 Simple Icons 的统一扁平化处理，SVG Porn 更倾向于保留品牌 Logo 的渐变、多色等原始视觉特征，适合需要展示"真实品牌外观"的场景。

高还原度 保留渐变色 CC0 协议 直接下载
svgl 500+ 品牌 MIT
svgl.app ↗

界面简洁、搜索迅速的品牌 SVG 查找工具，面向开发者日常快速取用场景设计。点击即可复制 SVG 代码或下载文件，效率极高。收录内容偏向科技和互联网品牌，特别是近年兴起的 SaaS 和 AI 产品。提供 API 接口，可编程化批量获取图标，也有 VS Code 插件。

快速取用 API 接口 VS Code 插件 SaaS / AI 品牌
Super Tiny Icons 475 品牌 MIT
GitHub ↗

以"极限压缩"为核心目标的品牌图标库——每个 SVG 文件大小均被压缩到 1KB 以内，最小的仅几百字节，是同类库体积的 1/10 到 1/20。代价是图标进行了必要的简化，省略了精细细节。但在 16–32px 的常规显示尺寸下视觉识别度依然良好。特别适合邮件模板、低带宽 Web 应用、IoT 设备界面等对文件大小敏感的场景。

每个 < 1KB 极限轻量 邮件模板适用 低带宽场景
theSVG 3,000+ 品牌 MIT
thesvg.org ↗

新兴品牌图标库，核心亮点是目前唯一大规模提供深色/浅色两套 Logo 变体的开源方案，解决了品牌 Logo 在深色背景下显示异常的长期痛点。同时提供了 MCP 服务器接口，支持 AI 工具链集成。随着深色模式成为标配，这一特性的价值正在快速上升。

深色/浅色双变体 Dark Mode 友好 MCP 接口 新兴库
World Vector Logo 3,000+ 品牌 非商业
worldvectorlogo.com ↗

大型品牌向量 Logo 聚合站，可按品牌名搜索并下载 SVG 文件，覆盖面广，包含许多小众品牌和传统行业品牌。注意：该站点的使用条款为非商业个人使用，与 Simple Icons 的 CC0 协议有本质区别，商业项目需谨慎。更适合用作设计参考和查找不常见品牌 Logo 的来源。

覆盖面广 含小众品牌 仅限非商业 设计参考

Openmoji 4,000+ Emoji CC BY-SA 4.0
